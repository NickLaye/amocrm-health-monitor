🚀 КАК ЗАПУСТИТЬ (БЫСТРЫЙ СПОСОБ):

1. Откройте терминал в этой папке

2. Запустите скрипт переключения:
   
   ./switch-design.sh new-demo

3. Запустите dev-сервер:
   
   cd client
   npm start

4. Откроется браузер с новым дизайном!


🔄 ДРУГИЕ ВАРИАНТЫ:

• Новый дизайн с реальными данными:
  ./switch-design.sh new-real
  cd client && npm start

• Возврат к старому дизайну:
  ./switch-design.sh old
  cd client && npm start

• Посмотреть текущий статус:
  ./switch-design.sh status

🎨 КАСТОМИЗАЦИЯ:

Все цвета настраиваются в одном файле:
  client/tailwind.config.js

Измените значения в секции colors:
  purple: {
    start: '#7B5FE8',  ← начало градиента
    end: '#9B6BE8',    ← конец градиента
    accent: '#7C3AED', ← акцентный цвет
  }
